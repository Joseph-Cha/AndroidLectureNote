### 개요

### 레이아웃 종류별 설명(Linear Layout, Relative Layout, Constraint Layout)
**Linear Layout**
* View가 추가 될 때 바로 다음 위치에서 추가가 된다.
* 즉 빈공간을 채워 넣는 방식으로 추가 된다.
